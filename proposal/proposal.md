# Pet Adoption Management System

## Project Description
This project will develop a Pet Adoption Management System database. The system will store and manage information about pets available for adoption, adopters, shelters and adoption applications.

The database will allow users to view pets available for adoption, submit adoption applications and track adoption status. It will also store information about shelters and staff responsible for managing the adoption process.

The project will include designing an ER diagram, creating database tables using SQL DDL statements, inserting sample data and performing SQL operations such as insert, select, update and delete. The system will also demonstrate how a Java application can interact with the database using JDBC.
